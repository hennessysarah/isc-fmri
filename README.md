# isc-fmri
scripts for intersubject correlation analysis (fmri, Hennessy et al., under rev: music-evoked nostalgia in older adults listening to "A day in the life" by the Beatles). Runs ISC groupwise (leave-one-out) analysis. 

1. beatles_ISC.py
2. beatles_ISC_ROI.py
3. permutation_test.py
4. permutation_ROI.py
5. SNS_and_ISC.py
6. threshold_and_niftify.py
